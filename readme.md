## Run Locally

Clone the project

```bash
  git clone https://github.com/savannahtech/adebayo-lychee.git
```

Go to the project directory (Frontend)

```bash
  cd lychee/frontend
```

Install dependencies

```bash
  npm install --legacy-peer-deps
```

Start the server

```bash
  npm run dev
```

To run the backend

```bash
  go to the project root directory
  cd lychee/backend
```

Install dependencies

```bash
  npm install
  npm install -g nodemon
```

Start the server

```bash
  nodemon index.js
```
